Internet

The Internet is a global network that connects millions of computers, allowing them to share information and communicate with each other.

Web

The web is a collection of pages on the Internet that you can visit using a web browser. These pages can include text, images, videos, and links to other pages.

How are the Internet and Web different?

The Internet is the network that connects computers globally. The Web is a collection of information you can access on the Internet using a web browser.

ASCII and Unicode

ASCII is a system that uses numbers to represent basic text characters like letters and symbols. It has 128 characters. Unicode is a bigger system that includes characters from almost all languages and symbols. It has over 140,000 characters.

What is UTF-8

UTF-8 is a way to encode text that uses 1 to 4 bytes for each character. It's compatible with ASCII and can represent any Unicode character.

URL

A URL is the web address you use to visit websites or access online resources.

Domain Name

A domain name is the web address you type to visit a website, like "example.com."

HTTP

HTTP is the basic way information is sent and received on the web.

HTTPS

HTTPS is a secure version of HTTP that encrypts the data sent between your browser and a website to keep it private and safe.

IP Address

An IP address is a unique number that identifies a device on the internet, like an address for your computer.

DNS

DNS is like a phone book for the internet. It changes website names into IP addresses so your browser can find them.

W3C Recommendations

W3C Recommendations are rules and guidelines made by the World Wide Web Consortium to help make websites work better and be more accessible for everyone.

XML, XHTML, HTML, HTML5

XML:A way to store and transport data, letting you create your own tags. XHTML:A stricter version of HTML that follows XML rules to make sure everything is well-formed. HTML:The standard language for creating web pages using various tags. HTML5:The latest version of HTML with new features for multimedia and interactive content.
